WordSMTP Simple SMTP Solution mailer. Easy & simple setup. All your WordPress / WooCommerce sending emails will be used your verified domain name. Know the status of your all emails through mail report.

== Description ==

WordSMTP simple SMTP solution to send emails from wordpress to recipient's inbox. All your WordPress emails will use your SMTP host/server to deliver your all emails to recipients. Know the status
of each sent email from your WordPress powered websites.

== Frequently Asked Questions ==

= What should I do first after activate the plugin? =
First you need to add your SMTP server information to send emails.

= Can I use my own hosting server SMTP? =
Yes you can use your own hosting server SMTP. You can ask your server administrator is there any restrictions / limit to send bulk emails.

= Can I use other bulk mailer server's SMTP like Sendinblue, Sendgrid, Amazon SES, Mandrill etc.? =
Yes you can use all of them cause they have support SMTP option including you can use any other custom Mailer server supporting SMTP. In case of other mailer server you should verify your
own domain / sender email address through their settings page. Cause other mailer server do not allow emails / deliver to inbox until sending email domain is verified / sending email address is verified.

= Can I check Mail Report? =
Yes you can check all your email report data with fail reasons and successfully delivered status of all emails.
